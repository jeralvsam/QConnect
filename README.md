QConnect is an emergency safety app built with Flutter, designed to make help accessible when it matters most. Users can report crimes, send SOS alerts, and access important helpline numbers in Kerala.

Features

Crime Reporting – Quickly report incidents to authorities

Emergency SOS – Instantly alert trusted contacts in case of danger

Helpline Directory – Access important helpline numbers in Kerala

Tech Stack

Flutter – Cross-platform mobile development

AI tools & resources – Assisted development and learning
Installation

Clone the repository:

git clone https://github.com/jeralvsam/QConnect.git


Navigate to the project folder:

cd QConnect


Install dependencies:

flutter pub get


Run the app:

flutter run
